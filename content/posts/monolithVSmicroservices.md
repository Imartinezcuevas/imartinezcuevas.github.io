# What is a Monolithic Architecture?

Software is traditionally designed using a monolith architecture, in which the entire program is constructed as a single, indivisible unit. Every component of the program is deployed and integrated tightly together in the design.
- This means that any changes or updates to the application require modifying and redeploying the entire monolith. 


# What is a Microservices Architecture?
A microservices architecture results in applications designed as a set of small independent services. Each one represents a business capability in itself. The services loosely couple with each other and communicate over the network.
- Each service is responsible for a single functionality or feature of the application and can be developed, deployed and scaled independently.
