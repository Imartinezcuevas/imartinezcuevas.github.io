# Definition
A model for enabling convenient, on-demand network access to a shared pool of configurable computing resources that can be rapidly provisioned and relased with minimal management effort or service provider interaction.

# Essential characteristics
- On-Demand self service: you can access cloud resources whenever required. There are no exceptions other than when services ara unable due to an outage or security breach.
- Broad network access: resources can be accessed throguth the network
- Resource pooling: consumers save costs hwen using a shared model. Computing resources are pulled to serve multiple consumers using a multi-tenant model. Cloud resources are assigned and reassigned according to demand without customers needint to concern themselves with the physical location of these resources.
- Rapid elasticity: you can increase or decrease resources as per your demand.
- Measured service: you only pay for what you use or reserve as you go.
