## What is a web app?
A web app, is a program stored on a remote server and delivered over the internet. A user interacts with the app using a browser. A web app requires three components:
1. A web server to manage raised request.
2. An app server to execute the requested task.
3. A database to store the information.

## What is an API?
An API is a software component that enables two app that are not connected with each other to communicate. APIs help create more flexibility for programmers and enable  them to request data from otherwise closed-off apps. APIs has standardized rules and functions that determine what data can be taken or modified within an app and how that process occurs.

There are several architectures software developers follow to create an API, buyt the most popular is Representational State Transfer (REST) or Simple Object Access Protocol (SOAP).
